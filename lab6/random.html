Правила игры:

  1. Для начала игры "Угадай Число" нажмите кнопку "Сыграть в игру"
  2. Выберите задуманное число из диапазона от 1 до 100
  3. Напишите выбранное вами число в поле для ввода
  4. Нажмите кнопку "Угадать" для проверки выбранного вами числа с загаданным
  5. Скорректируйте введенное вами значение в поле в соответствии с появившимся сообщением
  6. Нажмите кнопку "Сыграть ещё раз" в случае победы и желании начать игру заново
